Clinical Background

Thoracic diseases are among the most prevalent and high-stakes conditions in emergency and primary care. Conditions such as pneumonia, pulmonary edema, and pleural effusion can be life-threatening if missed, yet many share overlapping symptoms — shortness of breath, fever, chest discomfort — that make bedside differentiation unreliable.

Chest radiography remains the most widely used first-line imaging tool for thoracic disease, but radiologist availability is a bottleneck in high-volume settings. An AI screening assistant capable of simultaneously flagging multiple pathologies from a single X-ray could help triage patients more effectively and reduce diagnostic delay.

This project addresses that challenge directly: building a multi-label deep learning classifier capable of detecting 5 distinct thoracic conditions from a single chest radiograph with clinically meaningful sensitivity and specificity.

Dataset

CheXpert Plus (Stanford AIMI, 2026) is one of the largest publicly available chest radiograph datasets, comprising 223,462 radiographs from 187,711 studies across 64,725 patients. The dataset provides labels for 14 pathologies derived from radiology reports using automated NLP labeling.

Label uncertainty challenge: CheXpert's automated labeling pipeline produces three label states — positive (1), negative (0), and uncertain (u). Uncertain labels represent cases where the radiologist report was ambiguous. Handling these correctly is critical to model performance.

Preprocessing — U-Ones policy: Uncertain labels were mapped to positive (1), and missing labels to negative (0). This conservative approach reduces false negatives, which is preferable in a clinical screening context where missing a disease is more costly than a false alarm.

Train / Validation / Test Split:

  • Training: 70%
  • Validation: 15%
  • Test: 15%

Augmentation: Random horizontal flips and color jitter were applied during training to improve generalization and reduce overfitting. Low-quality and blurry images were filtered from the dataset prior to training. Images were uniformly resized for network input.

Pathologies detected (5-class multi-label):

Atelectasis Cardiomegaly Consolidation Edema Pleural Effusion

Model Architecture

I selected DenseNet121 as the backbone for this classifier. DenseNet (Densely Connected Convolutional Network) uses a unique architectural principle where each layer receives feature maps from all preceding layers via dense blocks connected by transition layers. This dense connectivity encourages feature reuse, reduces vanishing gradients, and substantially reduces the number of parameters compared to equivalent ResNet architectures.

DenseNet121 has been widely validated for chest X-ray classification tasks (notably in the original CheXNet paper from Stanford), making it a well-motivated choice for this dataset.

The model was configured for multi-label classification with a sigmoid output layer (one sigmoid unit per pathology class), allowing independent probability estimates for each condition from a single forward pass.

Weights: The model was initialized with ImageNet-pretrained weights, which provide strong low-level feature representations (edges, textures, shapes) that transfer effectively to medical imaging tasks — reducing overfitting on the label-noisy portions of the dataset.

Training & Methodology

Loss function: Binary Cross-Entropy (BCE) with logits, applied independently to each of the 5 output classes. A masked average loss was used to handle missing labels without penalizing the model for conditions not documented in a given study.

Optimizer: Adam with an initial learning rate of 0.0001.

Learning rate schedule: Reduce on Plateau — the learning rate was automatically reduced when validation loss stopped improving, allowing the model to converge more precisely in later training stages.

Early stopping (custom implementation): Rather than training for a fixed number of epochs, I implemented a custom early stopping rule based on the validation AUROC rather than validation loss. Training was halted automatically after 3 consecutive epochs where AUROC did not improve beyond the current best. This approach:

  • Prevents overfitting more precisely than loss-based stopping, since AUROC directly measures discriminative performance
  • Eliminates the need to manually tune epoch count — the model runs until it converges
  • Saved compute by terminating runs that had plateaued

Results

The DenseNet121 model achieved a mean validation AUROC of 0.9011, representing a 13.32% improvement over the DenseNet121 baseline (0.7952). Mean sensitivity improved by 13.57% and mean specificity by 13.52%.

Limitations & Honest Reflection

Despite strong aggregate metrics, several meaningful limitations remain:

  • Visually similar pathologies: Both DenseNet121 and EfficientNetB0 struggled to distinguish between pathologies with overlapping radiographic appearances — notably Consolidation vs. Atelectasis, and Edema vs. Pleural Effusion. These remain the lowest individual-class AUROC scores.
  • Label noise: CheXpert's NLP-derived labels contain inherent noise. The U-Ones mapping addresses uncertainty conservatively but introduces false positives in pathologies where uncertain labels are common.
  • Dataset bias: CheXpert Plus is drawn from a single health system (Stanford). Model generalization to populations with different imaging protocols, equipment, or disease prevalence distributions has not been validated.
  • No Grad-CAM validation yet: Visual inspection of attention maps to confirm the model focuses on clinically relevant regions (rather than spurious correlations) is planned but not yet completed.
  • Class imbalance: Some pathologies appear far less frequently than others, which can bias the model toward common conditions. Weighted loss functions or oversampling strategies could mitigate this.

Next Steps

  • Implement Grad-CAM visualizations to inspect model attention and validate clinical plausibility of predictions
  • Explore ensemble methods combining DenseNet121 and EfficientNetB0 to leverage complementary strengths
  • Investigate weighted BCE loss to improve performance on minority pathology classes
  • Extend evaluation to the held-out test set and compare against published CheXpert benchmark results
  • Explore 14-class full CheXpert label set rather than the 5-class subset
